## Projects

### 01. [Sutra Store Sales Analysis Dashboard (2025)](https://github.com/damaksharma/excel-projects/tree/main/sales-analytics/01_sutra-store-annual-sales-report-2025)
- **Tools Used:** Microsoft Excel, Pivot Tables, Pivot Charts, Slicers
- **Description:** Analyzed 15,000+ rows of retail sales data and built an interactive dashboard to explore sales trends, customer segmentation, and order channel performance.
- **Highlights:** 
  - Women contributed ~69% of total orders  
  - Amazon was the highest performing sales channel  
  - Maharashtra generated the highest revenue
